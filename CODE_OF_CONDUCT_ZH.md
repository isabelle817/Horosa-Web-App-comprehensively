# 行为准则

## 我们的承诺

我们希望星阙 Horosa Windows 仓库保持尊重、克制、可协作的交流环境。

贡献者、维护者与社区成员应当：

- 以善意沟通
- 关注行为和影响，而不是人身攻击
- 欢迎不同经验层级的参与者
- 在技术分歧中尽量使用证据和可复现事实

## 不可接受的行为

以下行为不可接受：

- 骚扰、威胁或歧视性语言
- 未经允许公开他人隐私信息
- 恶意钓鱼、反复搅乱讨论
- 发布密钥、凭据或敏感用户数据

## 处理方式

如果某种行为已经对项目或社区造成伤害，维护者可以删除内容、关闭讨论或限制参与。
