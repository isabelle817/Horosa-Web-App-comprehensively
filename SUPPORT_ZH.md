# 获取支持

## 一般使用问题

安装、配置、使用类问题请优先发到 GitHub Discussions：

- [GitHub Discussions](https://github.com/Horace-Maxwell/Horosa-Web-App-comprehensively-improved-Windows/discussions)

## 缺陷与回归

如果你有可复现的 bug 或明确的回归，请使用 GitHub Issues。

## 安全相关

安全问题不要开公开 issue。请按 [SECURITY_ZH.md](SECURITY_ZH.md) 报告。

## 说明

支持以 best-effort 方式提供。提问时请尽量附上：

- 当前版本号
- 你使用的是 web 端还是 Windows 桌面端
- 受影响的模块
- 清晰的复现步骤或截图
