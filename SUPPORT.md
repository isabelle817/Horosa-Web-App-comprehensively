# Support

## General usage help

For installation, configuration, or product usage questions, please use GitHub Discussions first:

- [GitHub Discussions](https://github.com/Horace-Maxwell/Horosa-Web-App-comprehensively-improved-Windows/discussions)

## Bugs and regressions

Use GitHub Issues when you have a reproducible bug report or a clear regression.

## Security-sensitive matters

Do not use public issues for security reports. Follow [SECURITY.md](SECURITY.md).

## Scope

Support is best-effort. When asking for help, include:

- the version you are using
- whether you are on web or Windows desktop
- the affected module
- clear reproduction steps or screenshots when relevant
